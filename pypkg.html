1、安装setuptools

利用setuptools来安装模块，所以如果没有setuptools的话，我们就需要安装一个setuptools,方法很简单，只需要在下方红圈另存为本地来， 让控制台转到ez_setup.zip的位置，然后运行

python ez_setup.py

如果出现安装出错，应该在lib文件夹里改变mimetypes.py,这样就算安装成功了。

注意：不要使用中文路径

同时偶们还会遇到这样错误的时候 需要去网站下载Microsoft Visual C++ Compiler for Python 2.7

2、安装模块和包

一般情况下我们可以简单的这样运行就可以,安装PIL模块

easy_install PIL

在国内我们应该使用国内的源，不然下载半天回报错 这是镜像的问题，需要使用国内的源，可找了半天都找不到配置文件在哪里。。。 继续挖。。。挖到运行这个

easy_install -i http://pypi.douban.com/simple/ numpy

因为无法找到配置文件在哪里，所以每次都得要带入这个国内源得链接，很多人都推崇豆瓣的源，也有别的源，像清华的源也可以，注意这里有在包前面有个空格

或者直接下载到本地安装

cd numpy-1.9.2
python setup.py install
